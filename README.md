 # Calculadora
 
 ## Projeto

 Desenvolvido no curso de Back-End da [Academia do Programador](https://www.academiadoprogramador.net) 2026.


## Introdução

Uma calculadora que permite realizar as quatro operações matemáticas, além da visualização do histórico de operações e da tabuada.


## Funcionalidades

- **Operações básicas:** Somas, subtrações, divisões e multiplicações com facilidade.

- **Tabuada:** A calculadora é capaz de gerar a tabuada de um número informado até outro número informado (o multiplicador).

- **Histórico:** A calculadora é capaz de armazenar na memória um histórico das operações realizadas anteriormente.


## Como utilizar o programa

1. Clone ou baixe os arquivos do repositório.
2. Abra o emulador de terminal da sua preferência e navegue até a pasta raiz do projeto baixado.
3. Utilize o comando abaixo para restaurar as dependências do projeto:
  ```
  dotnet restore
  ```
4. Em seguida, compile e execute o projeto com o comando:
```
  dotnet run --projectALTCalculadora.ConsoleApp
```

## Requisitos
- .NET 10.0 SDK
